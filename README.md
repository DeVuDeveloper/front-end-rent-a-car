- A terminal terminal
- A code editor
- React(follow the instructions based on your OS)

### Clone this repository

```bash
git clone https://github.com/Gegardus/book-a-car-frontend.git
```

### Move into the directory with

```bash
cd book-a-car-frontend
```

### Install linter

```bash
npm install
```

### Start server

```bash
npm start
```
